
## Let them eat Gherkin

### Install

```
cd <working-dir>
git clone https://github.com/listingslab-software/let-them-eat-gherkin
cd let-them-eat-gherkin
yarn && yarn start
```

### Run e2e

```
cd <working-dir>/let-them-eat-gherkin
yarn e2e
```

Should _just work_ &trade;
